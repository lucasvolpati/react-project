# App React

## Tecnologias Utilizadas

- React
- Typescript
- Vite
- Zustand
- Chackra UI
- Tan Stack Router
- SweetAlert

## Instalar as dependências do projeto

```
npm install
```

## Para executar o projeto em desenvolvimento

```
npm run dev
```

## Para fazer o buil do projeto

```
npm run build
```
